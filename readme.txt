Steps to Install and Run the Project:
Install Xampp : https://www.apachefriends.org/download.html
Download the project code and place it in htdocs folder in xampp folder.
start Apache and MySQL services from xampp control Panel.
Go to the browser type localhost/Library Management System
To access the database go to http://localhost/phpmyadmin and import the "id21063945_onlinehostedlibrary.sql" 
file given in the LMS folder.
Install VS Code or any other IDE to perform any changes locally in the project 
Now you are good to go.
